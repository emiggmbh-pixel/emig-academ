## Indikation & Anwendungsgebiet

Das **Vertebris Stenose**-System ist für die **vollendoskopische Dekompression** bei lumbaler und zervikaler Spinalstenose konzipiert. Es ermöglicht eine minimalinvasive Behandlung von Engstellen im Wirbelkanal, die durch degenerative Veränderungen entstehen.

| Indikation | Lokalisation | Zugangsweg |
|---|---|---|
| Lumbale Spinalstenose | L1 – S1 | Interlaminar bilateral |
| Zervikale Stenose | C3 – C7 | Interlaminar posterior |
| Laterale Stenose | Foraminal | Transforaminal |
| Hypertrophe Ligamenta flava | Lumbal / Zervikal | Interlaminar |

| Komponente | Außendurchmesser | Optik | Arbeitskanal |

| Vertebris SE Arbeitsendoskop | 7,2 mm | HD · 0° | 4,2 mm |

### Instrumente (Auswahl)

- **Kerrison-Stanzen** 1–4 mm, gerade und abgewinkelt
- **Mikrozangen** für Ligamentum-flavum-Resektion
- **Bipolare Elektroden** für Hämostase
- **Raspatorien** zur Dekortikation
- **Spülkanülen** (kontinuierlich / intermittierend)

## Technisches Verfahren

### Schritt 1 — Positionierung & Zugang

```
Patient: Bauchlage auf Wilson-Rahmen
Fluoroskopie: Markierung Zielwirbelsegment
Hautschnitt: 8 mm paramedian
Dilatation: Sequenziell Ø 4,2 → 6,8 mm
```

### Schritt 2 — Endoskopische Dekompression

Das Endoskop wird über ein Arbeitsportal eingeführt. Unter kontinuierlicher Kochsalzspülung erfolgt die schrittweise Dekompression:

1. Resektion des hypertrophen Ligamentum flavum
2. Ggf. Abtragung osteophytärer Anteile (Laminoplastie)
3. Bilaterale Dekompression durch Schwenken des Endoskops
4. Visualisierung des entlasteten Duraschlauchs

### Schritt 3 — Abschluss

- Spülung mit NaCl 0,9 % bis klares Retourflusskontrolle
- Schichtweiser Wundverschluss
- Keine Drainage erforderlich

## Klinische Vorteile

<Admonition type="tip" title="Klinischer Vorteil">
  Bilateraler Zugang durch unilateralen Schnitt — nur eine Hautinzision für beidseitige Dekompression.
</Admonition>

| Parameter | Endoskopisch (RIWOspine) | Offen chirurgisch |
|---|---|---|
| Hautschnitt | 8 mm | 40–80 mm |
| Blutverlust | < 30 ml | 100–400 ml |
| Stationärer Aufenthalt | 1–2 Tage | 3–5 Tage |
| Mobilisierung | ≤ 4 Stunden post-OP | 24–48 Stunden |
| Allgemeinanästhesie | Nicht zwingend | Erforderlich |

## Spülung & Irrigation

Das System arbeitet mit **kontinuierlicher Niederdruckspülung** (NaCl 0,9 %, Raumtemperatur):

- **Einfluss-Druck:** 30–50 mmHg
- **Flussrate:** 100–300 ml/min
- **Funktion:** Visualisierung · Hämostase · Neurolyse
- **Vorteil:** Nervenschutz durch hydrostatische Dekompression

> ⚠️ Bei übermäßigem epiduralen Druck → Druckreduktion sofort einleiten!
