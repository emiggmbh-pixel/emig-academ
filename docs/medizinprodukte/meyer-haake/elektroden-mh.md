## Übersicht

Das **MH-04 Elektroden & Zubehör**-Portfolio von Meyer-Haake umfasst alle Elektroden, Bipolarpinzetten, das BIO-CONE-System für Konusbiopsien sowie das gesamte Zubehör für den radioSURG® 2200. Das Sortiment wird kontinuierlich erweitert — zuletzt mit 3 neuen Elektroden (u.a. EELDRA91 für Blepharoplastiken).

<Admonition type="info" title="Kompatibilität">
  Alle MH-04 Elektroden sind ausschließlich für den Einsatz mit dem radioSURG® 2200 (MH-01) konzipiert. Für die meisten Anwendungsfälle sind sowohl autoklavierbare Mehrweg- als auch sterile Einwegelektroden verfügbar.
</Admonition>

## BIO-CONE — Konusbiopsie-System

Das **BIO-CONE** ist eine Spezialelektrode für die **Konusbiopsie an der Portio uteri** (Zervix-Konisation):

### Technisches Design

```
Form:          Kegelförmige Elektrode (Konus)
Besonderheit:  DREHBAR — kann rotiert werden während des Eingriffs
               MIT Stabilisierungsdornen zur Zervix-Fixierung
Vorteil:       Zervix wird fixiert → präzisere Konusgeometrie
Schnitt:       Radiofrequenz — schonend, ohne Verbrennungsartefakte
Histologie:    Beurteilbar bis Schnittrand — keine thermischen Artefakte
```

### Klinische Vorteile BIO-CONE vs. Messerkonisation

| Parameter | BIO-CONE (radioSURG®) | Messerkonisation (Cold Knife) |
|---|---|---|
| Thermische Artefakte | Minimal | Keine (aber mehr Blutung) |
| Blutung | Gering (Koagulation integriert) | Signifikant |
| Schnittrand-Beurteilbarkeit | Sehr gut | Optimal |
| Anästhesie | Lokal möglich | Spinal/Allgemein |
| Ambulant durchführbar | Ja | Nein |
| Zervix-Fixierung | Ja (Stabilisierungsdornen) | Manuell |

## Auswahlhilfe — Welche Elektrode für welchen Eingriff?

```
Schneiden präzise:   → Nadelelektrode oder Schlittelektrode
Flächige Exzision:   → Schlingenelektrode (rund/oval)
Koagulation:         → Ballektrode
Fulguration:         → Ballektrode (Abstand zum Gewebe)
Blepharoplastik:     → EELDRA91 (NEU)
Tonsillektomie:      → Tonsillektomie-Elektrode
Konisierung:         → BIO-CONE
Nervennähe:          → Bipolarpinzette
Gyäkologie (LEEP):   → Schlaufenelektrode gynäkologisch
Unterminierung:      → Hakenelektrode
```

## Aufbereitung

### Autoklavierbare Mehrwegelektroden

| Schritt | Parameter |
|---|---|
| Vorreinigung | Sofort nach Einsatz — enzymatisch |
| Ultraschall | 40 °C · 10 min |
| Spülung | Aqua dem. 3× |
| Inspektion | Isolierung intakt · kein Abbrand |
| Autoklav | **134 °C · Vorvakuum · 5 min** |

<Admonition type="danger" title="Isolierungsschäden">
  Elektroden mit beschädigter Isolierung dürfen NICHT mehr verwendet werden — Kurzschluss- und Verbrennungsgefahr. Sofort aussortieren und als Sperrware kennzeichnen (SOP-LOG-03).
</Admonition>

### Einwegelektroden

- ❌ Keine Wiederaufbereitung zulässig
- ❌ Keine Resterilisation
- Nach Einsatz: Entsorgung als medizinischer Abfall
