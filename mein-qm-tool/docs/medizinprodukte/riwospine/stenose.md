# Vollendoskopische Dekompression bei spinaler Stenose

Dieses Modul behandelt die Behandlung der lumbalen Spinalkanalstenose mittels der vollendoskopischen Technik von RIWOspine.

## Übersicht
Die lumbale Spinalkanalstenose ist eine degenerative Einengung des Wirbelkanals. Mit dem **VERTEBRIS Stenose** System wird eine gezielte, minimalinvasive Dekompression ermöglicht.

## Indikationen
* Zentrale Spinalkanalstenose
* Laterale Rezessusstenose
* Foraminale Stenose

## Technische Besonderheiten
* **Große Optik:** Spezielle Endoskope mit großem Arbeitskanal für effizientes Fräsen.
* **High-Speed-Burrs:** Knochenabtragung unter ständiger Sichtkontrolle.

## Lernziele
1. Verständnis der interlaminären Dekompression.
2. Korrekte Handhabung der Knochenfräsen.
3. Sicherheit im Umgang mit neuralen Strukturen unter endoskopischer Sicht.
